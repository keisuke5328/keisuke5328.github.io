Research at Autonomous Flight Systems Laboratory

Sept 2013 - Jun 2015

Conservative algorithms for automated collision awareness for multiple Unmanned Aerial Systems

As a research group, we developed a collision awareness algorithms for multiple UASs.
Publication: Conservative algorithms for automated collision awareness for multiple Unmanned Aerial Systems

My role was to generate unit tests for the software development kit created at University of Washington which we used to develop the software for the collision awareness systems.

UW-UAS

In this project, we constructed the UAS systems (hardware and software) for the laboratory. Using Skywalker 1900 as the hardware and Pixhawk, we implemented several add-on hardware and software.

I constructed the camera system for the image analysis onto the UAS. In addition to the camera systems, I modified the ground station software (Mission Planner) to output the sensor information of the UAS to a file as the they fly.
